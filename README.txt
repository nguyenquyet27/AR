# MARKER-BASED AUGMENTED REALITY 
___
This is a school project for CS 231.K11 of augmented reality based on markers with low-level imaging features.
___
## Overview

## Getting Started
We are still developing it and the current version only supports windows 10. So if you want to run it yourself, follow the instructions.

### Prerequisites
We assume that you already had python 3.x.x. If not, please install [python 3](https://www.python.org/downloads/) 
We recommend you using virtual environments.

### Installing
A step by step series of examples that tell you how to get a development env running.
```
pip install -r requirements.txt
```

## Library Versions
* [OpenCV](https://opencv.org/) - version: 4.1.2.3
* [Numpy](https://numpy.org/) - version: 1.17.4
* [PyOpenGL](http://pyopengl.sourceforge.net/) - version: 3.1.4

***!!!Note that** we haven't test with other versions of those libraries so please update your libraries to it the version that we mention. Big thanks*

## How to use
```
python main.py [-vb <video backend i/o>]
```
